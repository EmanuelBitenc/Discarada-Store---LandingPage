# Discarada Acessórios - Landing Page

Esta é a aplicação oficial (Landing Page) da marca **Discarada Acessórios**, construída com tecnologias modernas de performance web para proporcionar uma experiência fluida, rápida e luxuosa.

## 🚀 Tecnologias Utilizadas

- [Next.js (App Router)](https://nextjs.org/) - Framework React para SSR/SSG.
- [Tailwind CSS v4](https://tailwindcss.com/) - Estilização utility-first.
- [Framer Motion](https://www.framer.com/motion/) - Biblioteca para animações e micro-interações.
- **TypeScript** - Tipagem estática e segurança.

## 💻 Instalação e Execução Local

Pré-requisitos: Node.js (v18+)

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Execute o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver o resultado.

## 📁 Estrutura de Diretórios
- `src/app/` - Rotas, layout global e folha de estilos principal (`globals.css`).
- `src/components/` - Componentes modulares da UI (Hero, Navbar, Lojas, etc.).
- `src/data/` - Dados estáticos externalizados (ex: links de Lojas).
- `public/` - Assets estáticos da marca (imagens, logo e depoimentos).

## ✨ Deploy
O projeto está otimizado e pronto para deploy automático em provedores como a **Vercel**. Basta conectar seu repositório GitHub e realizar o deploy (Zero Configuration).
